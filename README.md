# NBT Jam!

A cross-platform Minecraft NBT file editor.

## Preview

![](./preview/1.png)

## Features

+ Parse / view / edit Minecraft [NBT format files](https://minecraft.fandom.com/wiki/NBT_format);
  - level.dat
  - `<player>.dat`
  - idcounts.dat
  - villages.dat
  - raids.dat
  - ...
+ Modern GUI preference;
